# &nbsp;　操弾 / Manipulate-Arrow
自ら放った矢を自由自在に操作することができます。<br><br>

## ◈ 対応バージョン<br>
  ・1.16 以上<br><br>
  
## ◈ 使い方
フォルダーに入れて /reload などすれば、放った矢が操れるようになっていると思います。<br>
無効化したければ /datapack コマンドから、データパックごと無効化しちゃってください。<br><br>
